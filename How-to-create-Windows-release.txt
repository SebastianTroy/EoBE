Windows release created using windeployqt.exe
-Go to projects
-Go to release build
-"Add build step"->"Custom process step"
-In the newly added step, set the following values:
    Command: %{Qt:QT_INSTALL_BINS}\windeployqt.exe
    Arguments: "%{CurrentRun:Executable:FilePath}"
    Working Directory: %{buildDir}
-When release build is built, all necessary Qt dependancies are copied into the build folder
-Pack .exe and dependancies into a zip (don't forget directories, don't copy .o files)
